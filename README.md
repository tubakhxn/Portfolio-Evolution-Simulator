# Portfolio Evolution Simulator

## What is this project about?
This project simulates the evolution of a financial portfolio using Monte Carlo methods, enhanced with machine learning. It generates hundreds of possible future portfolio paths, dynamically estimates drift and volatility using regression, and visualizes the results with stunning 3D animated and statistical plots.

**Features:**
- Monte Carlo simulation of portfolio paths
- ML-based dynamic drift/volatility estimation
- 3D animated visualization (matplotlib)
- Histogram, average, and rolling volatility plots
- Neon/dark theme styling

## How to fork and run
1. **Fork this repository** using the GitHub interface (top right "Fork" button).
2. **Clone your fork** to your local machine:
   ```
   git clone https://github.com/YOUR_USERNAME/Portfolio-Evolution-Simulator.git
   ```
3. **Install requirements**:
   ```
   pip install -r requirements.txt
   ```
4. **Run the project**:
   ```
   python main.py
   ```
